# API

There are autogenerated resource routes at **_/api/:entity_** for every [Entity](/docs/core-models/custom-entity/entity) that has the _"Has API"_ option enabled.

## Autogenerated API

- **GET** - Get all entity rows at **_/api/:entity_**.
- **POST** - Create rows at **_/api/:entity_**.
- **GET** - Get row at **_/api/:entity/:id_**.
- **PUT** - Update row at **_/api/:entity/:id_**.
- **DELETE** - Delete row at **_/api/:entity/:id_**.
